# Requirements
- Git
- Stow 
# Installation
### Install dependencies
- Debian
```sh
# apt install git stow
```
- Void
```sh
# xbps-install git stow
```  
- Clone the repository
```sh
$ cd ~
$ git clone https://github.com/Peelshine/dotfiles
```

- Setup symlinks with stow
```sh
$ cd ~/dotfiles
$ stow .
```
